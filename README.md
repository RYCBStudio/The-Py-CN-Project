## 我该怎么使用它？
1. 首先，下载Python（如果你没有的话，推荐3.10+）
2. 然后，下载本项目
3. 最后，（最好）在本项目文件夹下新建一个文件，后缀名为<i>.pycn</i>，文件名自取，最后运行项目中的<i>`main.py`</i>，按操作进行即可。
